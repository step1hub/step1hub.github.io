Copy this prompt, paste it into Claude or ChatGPT, then replace the last line with your copied table data:

Rules: - Date: the test date, formatted as MM/DD/YYYY - Questions: the number in the "Items" or "# Qs" column (integer only, no decimals) - Name: the test name or number exactly as shown (e.g. "Test 157", "Free 120") - One row per test session - Skip any rows where Items/Qs is blank, 0, or omitted - Skip any exam review rows or summary rows - No header row in output - No markdown, no code fences, no extra columns Output only the raw CSV. Example format: 03/01/2026,44,Test 201 02/28/2026,40,Test 200 Here is my data: [paste your copied table here]
